**Автор:** [Пендуров Фёдор]
**Дата создания:** [Май 2026]

## Описание

Password-Generator - GUI Приложение для генерации случайных паролей с настраиваемыми параметрами и сохранением истории.

### Возможности:
* Настройка длины пароля (4–64 символа)
* Выбор типов символов: цифры, буквы, спецсимволы
* Генерация пароля по нажатию кнопки
* Сохранение истории паролей в JSON
* Загрузка истории при запуске

## Использование

1. Запустите приложение: `python password_generator.py`
2. Настройте параметры:
   * Ползунок — выберите длину пароля
   * Чекбоксы — отметьте нужные типы символов
3. Нажмите «Сгенерировать пароль»
4. Пароль появится в поле вывода и добавится в историю

## Тесты

| Параметр | Ожидаемый результат |
|--------|------------------|
| Длина 8, только цифры | Пароль из 8 цифр |
| Длина 16, буквы + цифры | Смешанный пароль 16 символов |
| Без выбранных символов | Ошибка «Выберите хотя бы один тип» |
| Длина 3 | Ошибка «Длина должна быть от 4 до 64» |

## Зависимости

* Python 3.6+
* Стандартные библиотеки: `tkinter`, `random`, `json`, `os`, `datetime`
